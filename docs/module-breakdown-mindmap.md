## 基础设施层

- `🟩 运行环境`：项目环境、目录结构、资源目录、smoke 基线已形成
- `🟩 窗口管理`：窗口发现、绑定、聚焦、前台状态检查、多开实例绑定基础已具备
- `🟩 输入控制`：Win32 点击、按键、等待、审计记录、坐标换算链路已具备
- `🟩 配置体系`：环境、账号、角色、场景、UI、知识配置已成体系，边界校验已补齐
- `🟩 日志与证据`：运行日志、状态迁移、动作记录、截图、观察快照归档已具备
- `🟩 测试运行与调度`：smoke 基线稳定，当前全量 smoke 可通过

## UI 与感知层

- `🟥 主界面地图`：当前没有正式主界面地图识别链路
- `🟩 命名区域体系`：战斗相关区域已配置化，并支持 GUI 中调整和回写
- `🟩 模板识别`：模板目录、目录加载、OpenCV 模板匹配、独立识别测试页已具备
- `🟨 回合数字识别`：专用数字识别模块已落地，`1-9` 模板已可用，但还需继续做实机稳定性验证
- `🟥 颜色与像素识别`：还没有独立正式模块
- `🟨 弹窗与提示识别`：战斗提示和部分提示条已有基础模板，确认弹窗和异常提示未补全
- `🟨 状态识别`：`battle_scene / battle_round / battle_action_prompt / battle_skill_bar / battle_target_select / battle_settlement` 已拆模块，前四个更成熟，后两个偏占位
- `⬛ OCR 识别`：本项目内不再实现，统一改由外部 HTTP OCR 服务承担
