# 规则 JSON 样例

这些文件是从百科规则层继续收敛出来的第一批 JSON 样例。

用途：

- 验证知识层字段设计是否合理
- 作为后续正式配置的起点
- 供状态机、策略层、执行层对齐命名

边界：

- 当前仍是样例，不直接视为运行期配置
- 字段名稳定后，再决定是否迁入 `configs`
